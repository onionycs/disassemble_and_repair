# 拆卸的注意事项

1. 如果有拆机教程，先准备好相关的教程，from 贴吧，搜索引擎；
2. 要有一次处理完拆机后要做事情的计划，避免反复拆机造成的机械损坏；
3. 要通过GitHub做好document的备份；
4. 查看背板螺丝的设置；
5. 不要暴力拆卸；
